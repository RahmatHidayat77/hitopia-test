Kompleksitas waktu asimptotik algoritma : 
Karena **for** dieksekusi sebanyak n kali, sehingga komleksitas asimptotiknya
= O(1)+O(1)+O(1)+(n){O(1)+O(1)+O(1)+O(1)+O(1)}+O(1)
= O(1)+(n)O(1)+O(1)
= O(1) + O(n)
= O(max(1,N))
= O(n)
Jadi, kompleksitas waktu algoritma adalah O(n)
